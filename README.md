## Project Overview
This project analyzes global COVID-19 data to track infection trends, recovery rates, and regional severity using interactive Power BI dashboards.

## Tools Used
- Power BI (DAX, Power Query)
- Excel / CSV

## Dashboard Pages
1. Executive Summary
2. Trend Analysis
3. Country & Region Comparison
4. Recovery & Fatality Analysis
5. Growth & Risk Monitoring

## Dataset Source
Publicly available COVID-19 datasets from Kaggle and global health repositories.

## Key Insights
- Identified high-risk regions with rapid case growth
- Compared recovery and fatality rates across countries
- Analyzed time-based trends for decision-making
